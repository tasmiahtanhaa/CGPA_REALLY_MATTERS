# In IUT, CGPA Matters

A modular C project for calculating CGPA.

## Linux

Build:
```
chmod +x make.sh
./make.sh
```

Test:
```
chmod +x test.sh
./test.sh
```

Run:
```
chmod +x calculator.out
./calculator.out
```

## Windows (PowerShell)

Build:
```
.\make.ps1
```

Test:
```
.\test.ps1
```

Run:
```
.\calculator.exe
```
